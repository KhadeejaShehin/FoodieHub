# Restaurant Management System

A full-stack Restaurant Management and Online Ordering System built using Django that allows users to browse menu items, add them to a cart, and place orders through a smooth checkout process. The application is designed to simplify restaurant operations while providing a user-friendly online ordering experience.

---

##  Features

- User registration and login
- Category-based menu listing (Breakfast, Lunch, Dinner, Beverages)
- Add to cart and order placement
- Order summary and tracking
- Admin dashboard for managing menu listing and orders
- Responsive UI for desktop 

---

##  Technologies Used

- **Backend:** Python, Django  
- **Frontend:** HTML, CSS, Bootstrap, JavaScript  
- **Database:** SQLite  
- **Tools:** Git, GitHub
